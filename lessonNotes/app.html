Inline vs Block
Inline - ends a line and wraps text to the next one
a, span, br, img, strong are all inline tags\

Block - creates a box around the text, has a height and width
div, b, form are block tags
span and div both hold content - difference is span is inline and div is block

Lesson 3 Notes

CSS
CSS (Cascading Style Sheets) are used to apply style to webpages. Multiple sheets can be used for a single page and the term cascading refers to the concept of inheritance.

Inheritance: Ancestor -> descendant relationship.
Descendants inherit characteristics from ancestors

CSS allows programmers to avoid repetition of code. This will help increase readability, decrease load and debugging times, and more. It also saves time when writing code.

selectors - identify the element you want to apply styles to.

The line of code that says background-color : black; is a declaration, background-color is a property and black is the value of that property.

You can use classes to apply styles to only certain instances of an element.

Ways to apply CSS
Seperate sheet linked in <head> - Use for larger projects
In <head> as a <style> tag - Use for smaller projects
Inline with HTML - BAD IDEA Will come across from time to time but leads to lots of code repetition
